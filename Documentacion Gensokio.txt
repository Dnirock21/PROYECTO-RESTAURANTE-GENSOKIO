Documentación del Proyecto: Restaurante Gensokio
1. Introducción
Restaurante Gensokio es un proyecto dedicado a la promoción y presentación de la gastronomía tradicional japonesa. Este restaurante ofrece a los clientes una variedad de recetas auténticas de la cocina japonesa, destacando la frescura y tradición en cada plato.
El objetivo principal del proyecto es desarrollar una plataforma que permita a los usuarios explorar el menú del restaurante, conocer los ingredientes de cada receta de manera sencilla y eficiente. El proyecto se enfoca en brindar una experiencia de usuario agradable y fluida.

3. Características del Proyecto:
El proyecto incluye las siguientes funcionalidades principales:
•	Visualización de recetas japonesas: Los usuarios pueden navegar por una amplia variedad de recetas tradicionales japonesas, cada una con una descripción detallada, lista de ingredientes y pasos de preparación.
•	Diseño intuitivo y responsivo: La plataforma está diseñada para adaptarse al tamaño de la pantalla de las computadoras de escritorio, garantizando una experiencia de usuario óptima.

4. Estructura del Proyecto:
l.      proyecto se organiza en tres componentes principales:
1.	Base de datos: Gestionada por Daniel Andres Cobaleda Montaño, almacena toda la información relacionada con los platos, recetas, ingredientes.
2.	Backend: Desarrollado por Karen Julieth Fajardo Granada, el backend conecta la base de datos con el frontend y maneja las operaciones como el registro de usuarios y otras funcionalidades del servidor.
3.	Frontend: Creado por Juan Diego Sanchez Castaño, el frontend se encarga de la presentación visual del sitio web y la interacción con los usuarios. Usa tecnologías modernas para ofrecer una experiencia de usuario fluida.

5. Tecnologías Utilizadas
El equipo ha utilizado las siguientes herramientas y tecnologías para desarrollar el proyecto:
•	Frontend: HTML, CSS, JavaScript
•	Backend: Node.js, Express.js
•	Base de datos: MySQL
•	Control de versiones: Git y GitHub para la gestión del código
•	Despliegue: Se prevé implementar la aplicación en un servidor web o plataforma en la nube.

6. Conclusión
Restaurante Gensokio es un proyecto integral que busca ofrecer a los usuarios una plataforma cómoda para explorar la rica tradición culinaria japonesa, facilitando el acceso a recetas. Gracias al trabajo coordinado del equipo, el proyecto está avanzando de manera exitosa hacia su implementación final.
